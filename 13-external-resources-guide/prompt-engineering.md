## Key Techniques Summary

| Technique | When to Use |
|-----------|-------------|
| **Be clear and direct** | Always - explicit > implicit |
| **Use XML tags** | Complex prompts with multiple sections |
| **Few-shot examples** | When showing is easier than telling |
| **Chain of thought** | Complex reasoning tasks |
| **Role prompting** | Setting persona and tone |
| **Prefilling** | Controlling output format |
| **Prompt chaining** | Multi-step workflows |
