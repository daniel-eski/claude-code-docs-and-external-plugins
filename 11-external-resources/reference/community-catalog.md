## Adding Skills from This List

To download any of these skills:

```bash
cd ../tools/
./fetch-skill.sh https://github.com/<author>/<repo> ../extended-skills/<category>/<skill-name>
```

Note: Some repositories may be private, renamed, or removed. Check the URL before fetching.
