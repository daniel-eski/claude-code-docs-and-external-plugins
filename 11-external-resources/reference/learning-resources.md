## Practice Projects

Suggested starter projects for skill development:

1. **Simple utility skill**: File format converter
2. **API integration skill**: Weather or stock price lookup
3. **Workflow skill**: Git branch management
4. **Document skill**: Report template generator
5. **Testing skill**: Test data generator

Start with the `template` in `core-skills/skill-authoring/` and iterate.
